h1. Dummy Text Generator Plugin

Source code of the intellij IDEA "Dummy Text Generator" plugin: http://plugins.jetbrains.com/plugin?pr=&pluginId=7216
This is a plugin for the various intellij IDEs, working in IDEA, PhpStorm, AppCode, etc.



h2. Description

The plugin adds a button to generate dummy text to the main toolbar.
Randomly generated dummy text can than be inserted with a click, text selections are being replaced
by dummy text of approximately the same length.

The plugin configuration allows to choose from several genres:

* Fake Latin
* Medical Drama
* Pirate Lingo
* Science Fiction
* Esoteric Wisdom
